# Music Instrument Randomizer (Python Script)

A simple Python console application designed to inspire your musical journey by randomly selecting an instrument from a wide variety of categories. Perfect for musical inspiration, random challenges, or just having some fun!

## Features

- **Wide Instrument Pool:** Includes a rich selection of string, keyboard, brass, woodwind, percussion, and other unique instruments.
- **Interactive Loop:** Allows continuous drawing of new instruments until the user decides to exit.
- **Theatrical Pauses:** Uses `time.sleep()` to simulate "thinking" before revealing the instrument.

## How to Run

### Prerequisites

You only need Python 3 installed on your system.

### Running the Script

1.  **Save the Code:** Ensure the provided Python code is saved as `script.py` (or a similar name) in your local machine.
2.  **Open Terminal:** Navigate to the folder where the file is saved.
3.  **Execute:** Run the script using the Python interpreter:

```bash
python script.py
```
